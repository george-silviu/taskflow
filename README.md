# Taskflow

## 📖 About

> A minimalist task management project built to practice the essential concepts of Angular

## ✨ Features

- 📋 View a user tasks
- ✅ Create tasks
- 🎯 Mark tasks as completed
- 📱 Responsive design for mobile and desktop
